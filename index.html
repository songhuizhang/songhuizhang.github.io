秋，腰，自律

伤痛

最近半个月的时间腰尽然连续扭伤两次，让年仅25岁的我想到一首歌《往事只能回味》。

歌词写的好时光已逝永不回,往事只能回味虽然这首歌是说两小无猜，青梅竹马的往事恋情。

但是此前此景此腰让我有着另一番感受:

大学的时候正值青春年华，那会儿的我是不知疲惫为何物。（身体和心理）

现在毕业两年时间，体重涨了20斤，本该充满活力的身体也应为久坐和姿势的问题变得容易受伤。

有句名言：你才25岁，你可以成为任何你想成为的人。

我一直对此话深信不移，人的一生有着无限的可能性，只要努力就一定可以比现在过的更好的。

但是，还有一句名言身体是革命的本钱

现在的我体重微胖、腰肌劳损、作息不规律，没有任何革命的本钱。现在都在说90后已经步入养身大军，这月的两次受伤让我真心觉得原来不在意的事情，尽然在慢慢侵蚀着我的身体。

自律

人都说不要常立志要立长志。

我喜欢追求自由，但是现在看似自由的我正在被身体的伤痛，口腹的欲望所缠绕。有一期锵锵三人行中，窦文涛说过这么一句话：“只有自律的人，才是真正自由的人。”。

自律是一种生活的态度，看似给生活加了许多的条条框框来约束自己。但是，让你习惯后你会发现这些条条框框不是对自我的约束，而是一种促使自己实现自我目标的动力。

我的目标是人生的自由，这需要有一个健康的身体，有着足够的金钱；在拥有这些的前提下，才能去尝试着追寻自我的自由。

结语：希望明天的自己会满意现在自己走的每一步，现在的自己更要努力不让明天的自己失望！！

《JavaScript高级程序设计》---基本概念

  • 语法
  • 数据类型
  • 流控制语句
  • 函数

语法

区分大小写、标识符以驼峰法命名、注释的用法、以分号结尾的语句、相关关键字保留字、变量

数据类型

ES5有5中简单数据类型：undefined，null，Boolean，number， string；有一种复杂数据类型：Object；

typeof

是操作符，返回值为字符串，一般可以用来区分函数和其他对象；

undefined

一般来说字面量的undefined用于比较

null

null表示一个空指针，所以typeof null返回的是“object”；如果定义变量准备保存对象，那么最好可以将其初始化为null。

boolean

数据类型 转为true的值 转为false的值
boolean true false
string 任何非空字符串 “”空字符串
number 任何非零数字 0和NaN
object 任何对象 null
undefined 不适用 undefined

以上转换规则在流程控制中很重要

number

浮点数值计算舍入误差的问题,下面是原生解决办法：
parseFloat((0.1 + 0.2).toFixed(10))

NaN

任何涉及NaN的操作都返回NaN，NaN与任何值都不想等，包括NaN本身。

string

除了null和undefined没有，几乎所有值都有toString()方法。

object

对象其实是一组数据和功能的集合

object的每个实例都具有下列属性和方法：

  • constructor 保存用户当前创建对象的函数
  • hasOwnproperty() 用于检查给定的属性在当前对象实例中是否存在
  • isPrototypeOf() 用于检查传入的对象是否是另一个对象的原型
  • propertyIsEnumerable() 用于检查给定的属性是否能够使用for-in语句来枚举
  • toLocaleString() 返回对象的字符串表示
  • toString() 返回对象的字符串表示
  • valueOf() 返回对象的字符串、数值或布尔值表示

操作符

一组用于操作数据值的操作符

  • 一元操作 ++ –
  • 位操作
  • 布尔操作
    1. 逻辑非，可以用于将一个值转为气对应的布尔值
    2. 逻辑与： 逻辑与的短路操作，不能在逻辑与中使用未定义的值
    3. 逻辑或： 逻辑或的短路操作，在赋值语句中一般使用逻辑或提供后备值
  • 乘性操作
    1. 乘法 *
    2. 除法 /
    3. 求模 %
  • 加性操作
    1. 加法 计算规则需要注意
    2. 减法
  • 关系操作 < > <= >=,返回布尔值
  • 相等操作 == != 先转换再比较， === !== 直接比较
  • 条件操作 boolean? true : false;
  • 赋值操作 =
  • 逗号操作 可以在一条语句中执行多个操作。

    语句

    if 语句

    推荐写法，一代码块包裹

do-while 语句

推荐用法：至少要执行一次的情景

while 语句

前测试，不满足条件不会执行

for 语句

for-in 语句

为了防止枚举对象为undefined或null，所以使用前应先判断

label 语句 （较少使用）

break 和 continue 语句

break和continue用于在循环中精确的控制代码的执行

break是直接推出循环，而continue也是推出循环，但是它会从循环顶部继续执行

with 语句 （不使用）

switch 语句

函数

函数可以封装任意多条语句，而且可以在任何地方任何时候调用。

在函数定义式不必指定返回值，任何函数在任何时候都可以通过 return 语句来实现返回值；

推荐一个函数要么一直有返回值，要么一直没有返回值

在函数内部可以使用argumants对象来获知有多少个参数传入。

上述为JavaScript的基本概念，其中函数相关在后续再详细展开。

你好,Hexo

初次尝试使用hexo搭建个人博客，hexo是一个使用Markdown为语法的博客框架。

网上有许多的教程关于如何使用hexo+github page来做博客的教程。这里我简述一下过程

  1. 本地全局安装hexo脚手架npm install hexo -g
  2. 使用脚手架新建一个blog项目hexo init
  3. 新建一个github账号，并创建一个username.github.io为名的仓库
  4. 配置ssh关联github账号
  5. 设置_config.yml发布配置，将博客内容同步至github
  6. 通过username.github.io访问

上述流程不是教程，只是自己用来回忆的索引

可扩展项：

  • 主题的变更
  • 域名的绑定
