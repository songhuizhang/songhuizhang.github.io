正则表达式

发布 : 2018-12-01 分类 : JavaScript 浏览 :

RegExp对象

在JavaScript中RegExp 对象有两种模式：字面量，构造函数

···
var reg1 = /\bhello\b/g // 字面量
var reg2 = new RegExp(‘\bhello\b’,’g’); // 构造函数
···

元字符

正则表达是由两种基本字符类组成： 愿义字符 元字符

  • 愿义字符：表示原本意思的字符
  • 元字符：表示不是原本意思的字符

愿义字符hello

元字符 \d

如果要匹配元字符本身，则需要使用转义符\

元字符 意义
$ 匹配输入字符的结尾位置
() 标记一个字表达式的开始和结束位置
* 匹配前面的子表达式零次或多次
+ 匹配前面的子表达式一次或多次
. 匹配除换行\n外的任何单字符
[] 标记一个中括号表达式的开始
{} 标记限定符表达式的开始
指明两项之间选一项
? 匹配前面的子表达式零次或一次
^ 匹配输入字符串的开始位置
\cX 匹配由x指明的控制字符
\f 匹配一个换页符
\n 匹配一个换行符
\r 匹配一个回车符
\s 匹配任何空白字符，包括空格、制表符、换页符等等
\S 匹配任何非空白字符。
\t 匹配一个制表符
\v 匹配一个垂直制表符
