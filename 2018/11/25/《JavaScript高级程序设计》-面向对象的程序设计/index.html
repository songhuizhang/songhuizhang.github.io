《JavaScript高级程序设计》---面向对象的程序设计

发布 : 2018-11-25 分类 : JavaScript 浏览 :

面向对象语言都有一个标志，那就是类。在JavaScript中没有类的概念，因此它实现面向对象的和传统的面向对象语言有所不同。

理解对象属性

创建自定义对象

方法一：通过Object创建一个实例，再为它添加属性和方法；

var person = new Object();
person.name = “winnie”;
person.age = 26;
person.job = “Font Enginner”;

person.sayName = function(){
alert(this.name);
}

方法二：通过字面的方法创建对象

var person = { name: "winnie", age: 26, job: "Font Enginner", sayName: function(){ alert(this.name); } }

属性类型

ECMAScript中有两种属性：数据属性和访问器属性

数据属性

包含一个数据值的位置，这个位置可以读取写入，它有4个描述行为的特征

  • [[Configurable]]:表示是否具有可以删除和可修改的特性，默认值：true
  • [[Enumerable]]:表示是否具有可以通过for-in返回的属性，默认值：true
  • [[writable]]:表示包含可以修改的属性的值，默认值：true
  • [[value]]:表示包含这个属性的数据值，默认值：undefined

修改默认属性特性，需要通过Object.defineProperty()方法，但是当[[Configurable]]为false后，就不能在修改属性特征了；


var person = {};
Object.defineProperty(person,”name”,{
wirtable: false,
value: “Winnie”
})

person.name // Winnie
person.name = “章三”; // 不可修改
person.name // Winnie

访问器属性

不包含数据值，只包含了一对getter和setter函数，再读取访问器属性时调用了getter，写入时调用了setter，访问器属性有如下四个特征

  • [[Configurable]]:表示是否具有可以删除和可修改的特性，默认值：true
  • [[Enumerable]]:表示是否具有可以通过for-in返回的属性，默认值：true
  • [[Get]]:读取属性时调用的函数，默认值：undefined
  • [[Set]]:写入属性时调用的函数，默认值：undefined

访问器属性不能直接定义，需要通过Object.defineProperty()来定义


var book = {
_year: 2004,
edition: 1
};

Object.defineProperty(book,”year”,{
get: function(){
return this._year;
},
set: function(newValue){
if (newValue > 2004) {
this._year = newValue;
this.edition += newValue - 2004;
}
}
})
