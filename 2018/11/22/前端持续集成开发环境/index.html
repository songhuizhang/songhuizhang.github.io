前端持续集成开发环境

发布 : 2018-11-22 分类 : 开发流程 浏览 :

前言

在掘金上看见基于 Docker 打造前端持续集成开发环境这篇文章，就像尝试一个基于docker的前端容器化部署方案。
我们公司现在也是基于jenkins+docker的代码容器化部署方案。

目标：

  1. 代码无需在本地构建

  2. 只需将代码推上 Github ，自动构建 -> 部署

  3. 版本易管理，可轻松回退版本

前置条件

daocloud账号+github账号+个人服务器 基于daocloud的在线解决方案适用于个人开发者，生产开发的话介于安全性考虑需要进行本地化部署（本文不涉及）

本地项目添加dockerfile

FROM node:6.10.3-slim RUN apt-get update \ && apt-get install -y nginx WORKDIR /app COPY . /app/ EXPOSE 80 RUN npm install \ && npm run build \ && cp -r dist/* /var/www/html \ && rm -rf /app CMD ["nginx","-g","daemon off;"]
